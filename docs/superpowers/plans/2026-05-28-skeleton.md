# Skeleton (Build Step 1) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ape Intel loads as a Firefox MV3 extension on Scalable Capital security pages, parses the ISIN from the URL, and renders a Badge attached to `document.body` that prints the ISIN.

**Architecture:**
- Vite + `@crxjs/vite-plugin` builds an MV3 extension. A content script auto-injects on the Scalable security URL pattern and mounts a Preact `Badge` component into `document.body` (not into Scalable's React tree — per ADR-0003).
- All pure logic (URL parsing, ISIN validation) lives in `src/lib/` and is unit-tested with Vitest. The content script wires DOM + Preact only.
- No data fetching yet. No SPA-nav handling yet (deferred to a later step — Step 1 demo is hard-reload per Asset).

**Tech Stack:** TypeScript, Vite, `@crxjs/vite-plugin`, Preact, Vitest, Firefox MV3.

## Task 2: ISIN parser with TDD

**Files:**
- Create: `src/lib/isin.ts`
- Create: `src/lib/isin.test.ts`

ISIN format reference: 12 chars, `^[A-Z]{2}[A-Z0-9]{9}[0-9]$`. We do not implement the Luhn-style check digit verification in v1 — pattern validation is enough to reject obvious garbage; a wrong-but-well-formed ISIN will simply fail downstream lookup.

- [ ] **Step 1: Write the failing tests**

`src/lib/isin.test.ts`:

```ts
import { describe, expect, it } from "vitest";
import { isValidIsin, parseIsinFromUrl } from "./isin";

describe("isValidIsin", () => {
  it("accepts a well-formed ISIN", () => {
    expect(isValidIsin("US0378331005")).toBe(true);
    expect(isValidIsin("DE0007164600")).toBe(true);
  });

  it("rejects wrong length", () => {
    expect(isValidIsin("US037833100")).toBe(false);
    expect(isValidIsin("US03783310055")).toBe(false);
  });

  it("rejects bad country code (must be 2 letters)", () => {
    expect(isValidIsin("1S0378331005")).toBe(false);
    expect(isValidIsin("u50378331005")).toBe(false);
  });

  it("rejects non-digit check digit", () => {
    expect(isValidIsin("US037833100X")).toBe(false);
  });

  it("rejects empty / nullish", () => {
    expect(isValidIsin("")).toBe(false);
    // @ts-expect-error null is not a string
    expect(isValidIsin(null)).toBe(false);
  });
});

describe("parseIsinFromUrl", () => {
  it("extracts isin from Scalable security URL", () => {
    const url =
      "https://de.scalable.capital/broker/security?isin=US0378331005&portfolioId=abc";
    expect(parseIsinFromUrl(url)).toBe("US0378331005");
  });

  it("returns null when isin query param is missing", () => {
    expect(
      parseIsinFromUrl("https://de.scalable.capital/broker/security?foo=bar"),
    ).toBeNull();
  });

  it("returns null when isin query param is malformed", () => {
    expect(
      parseIsinFromUrl(
        "https://de.scalable.capital/broker/security?isin=not-an-isin",
      ),
    ).toBeNull();
  });

  it("returns null for non-Scalable URLs", () => {
    expect(
      parseIsinFromUrl("https://example.com/broker/security?isin=US0378331005"),
    ).toBeNull();
  });

  it("returns null on unparseable URL", () => {
    expect(parseIsinFromUrl("not a url")).toBeNull();
  });
});
```

- [ ] **Step 3: Implement minimal `src/lib/isin.ts`**

```ts
const ISIN_PATTERN = /^[A-Z]{2}[A-Z0-9]{9}[0-9]$/;
const SCALABLE_HOST = "de.scalable.capital";
const SCALABLE_PATH = "/broker/security";

export function isValidIsin(value: unknown): value is string {
  return typeof value === "string" && ISIN_PATTERN.test(value);
}

export function parseIsinFromUrl(rawUrl: string): string | null {
  let url: URL;
  try {
    url = new URL(rawUrl);
  } catch {
    return null;
  }
  if (url.hostname !== SCALABLE_HOST) return null;
  if (url.pathname !== SCALABLE_PATH) return null;
  const candidate = url.searchParams.get("isin");
  return isValidIsin(candidate) ? candidate : null;
}
```

## Task 3: MV3 manifest

**Files:**
- Create: `manifest.config.ts`

Firefox MV3 quirks:
- Must declare `browser_specific_settings.gecko.id` or AMO signing fails.
- Content-script `matches` patterns: Scalable's URL uses query params, which `matches` ignores anyway — pattern matches on origin + path, so `https://de.scalable.capital/broker/security*` is correct. `?isin=…` filtering happens in the content script itself.
- `run_at: document_idle` so we mount after Scalable's initial render.

- [ ] **Step 1: Create `manifest.config.ts`**

```ts
import { defineManifest } from "@crxjs/vite-plugin";

export default defineManifest({
  manifest_version: 3,
  name: "Ape Intel",
  version: "0.0.1",
  description:
    "Community sentiment + news panel for Scalable Capital security pages.",
  browser_specific_settings: {
    gecko: {
      id: "ape-intel@lmueller.dev",
      strict_min_version: "121.0",
    },
  },
  content_scripts: [
    {
      matches: ["https://de.scalable.capital/broker/security*"],
      js: ["src/content/index.tsx"],
      run_at: "document_idle",
    },
  ],
});
```

## Task 6: Content-script entry — mount Badge on Scalable security pages

**Files:**
- Create: `src/content/index.ts`

The content script:
1. Runs at `document_idle` on Scalable security pages (filtered by manifest).
2. Calls `parseIsinFromUrl(location.href)` — guards against the manifest matching a page whose query string lacks `?isin=…`.
3. If valid, creates a host `<div>` appended to `document.body` and renders `<Badge />` into it with Preact.
4. If invalid/missing, exits silently (matches Coverage state handling later; for Step 1, no badge is correct behaviour on an unsupported URL).

- [ ] **Step 1: Implement `src/content/index.tsx`** (TSX because the file uses JSX)

```tsx
import { render } from "preact";
import { Badge } from "./Badge";
import { parseIsinFromUrl } from "../lib/isin";

const HOST_ID = "ape-intel-host";

function mount(isin: string): void {
  if (document.getElementById(HOST_ID)) return;
  const host = document.createElement("div");
  host.id = HOST_ID;
  document.body.appendChild(host);
  render(<Badge isin={isin} />, host);
}

const isin = parseIsinFromUrl(location.href);
if (isin) mount(isin);
```
